# PORTFOLIO
Kshitij Khilari | Personal Portfolio A sleek, modern, and fully responsive single-page personal portfolio website built to showcase my academic journey, technical skills, projects and achievements.

Overview This portfolio is designed with a dark, cinematic aesthetic, featuring smooth scroll animations, glassmorphism UI elements and interactive hover states. It serves as a digital resume and a centralized hub for my work as a BCA student and developer.

Features Modern Dark UI: A cinematic dark theme with subtle gold (#c8a96e) and purple (#7c6fff) accents.

Immersive Animations: Custom SVG noise overlay for texture.

Floating background grids and ambient glowing orbs in the hero section.

Smooth entry animations using the JavaScript IntersectionObserver API.

Responsive Design: Fully fluid layout using CSS Grid and Flexbox, ensuring a perfect view on desktops, tablets and mobile devices.

Interactive Skill Bars: Dynamic width transitions on hover to showcase technical proficiency.

Glassmorphism Navbar: A sticky navigation bar with a blurred backdrop for easy access to different sections.

Tech Stack This project is built using pure vanilla web technologies no heavy frameworks or external libraries required.

HTML: Semantic structure and content layout.

CSS: Styling, variables (:root), complex keyframe animations, responsive clamp functions, and grid layouts.

JavaScript: Intersection Observers for scroll-triggered opacity fades and dynamic event listeners for hover states.

Fonts: Hosted via Google Fonts ('Playfair Display' for headings, 'DM Mono' for body text and code elements).

Project Structure This is a lightweight, zero-dependency project contained entirely within a single file for maximum portability and fast loading.

index.html - Contains all HTML structure, embedded CSS styling and JavaScript logic.

Usage & Installation Since this is a static single-page application, no build steps or package managers are required.
